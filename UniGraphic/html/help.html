键盘快捷键 (Keyboard shortcuts)

快捷键说明
ctrl + 滚轮缩放画布
ctrl + +/-缩放画布
ctrl + 左键多选
空格 + 左键拖动画布
esc切换鼠标状态为选择
enter查看/编辑选中的节点/连线
delete删除选中的节点/连线
ctrl + s保存
← ↑ ↓ →移动节点/画布
